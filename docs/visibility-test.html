BP Animate - Visibility Detection Test

This demo tests how BP Animate handles elements that are in the viewport but hidden via different CSS methods.

Expected behavior: All hidden elements should get bp-is-hidden class, NOT bp-is-visible or bp-is-animating.

Test 1: opacity: 0

What this tests: Elements with opacity: 0 are technically in the viewport but invisible to users.

Expected: Should get bp-is-hidden class (not bp-is-visible or bp-is-animating).

Note: Elements with display: none won't be detected by Intersection Observer, so they won't trigger any classes. This is expected behavior.

What this tests: Elements with visibility: hidden are in the layout but invisible.
